## 🌍 About Synkora

Synkora ERP is a modern, modular and open-source ERP built specifically for SACCOs, Credit Unions and Cooperative Financial Institutions.

Unlike traditional ERP systems, Synkora is designed around **Features** and **Plugins**, allowing organizations to install only what they need.

Our vision is simple:

> Build the WordPress of SACCO ERP systems.

---

## ✨ Vision

To create the world's most customizable open-source SACCO ERP platform that empowers financial cooperatives through modern technology.

- Market API
- Currency Exchange API

# 🤝 Contributing

We welcome everyone.

Please **DO NOT** commit directly to `main`.

Workflow

```
Fork Repository

↓

Create Branch

↓

Develop

↓

Pull Request

↓

Review

↓

Merge into develop

↓

Release

↓

main
```

Example

```
git checkout develop

git checkout -b feature/member-registration
```
